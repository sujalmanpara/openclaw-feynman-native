# Session Log

Use this workflow to write a durable session log capturing completed work, findings, open questions, and next steps.

## Trigger phrases
- log this session
- save session notes
- write up what we did
- research diary entry

## Deliverable
- Session log: `notes/session-log-YYYY-MM-DD.md`

## Execution contract

### Required content
- summary of what was done
- strongest findings or decisions
- open questions and unresolved risks
- concrete next steps
- references to any artifacts written during the session
- direct source URLs for external claims that matter

### Format
```md
# Session Log — [date]

## Work Done

## Key Findings

## Open Questions

## Next Steps

## Artifacts Created
- [list of files written this session]
```

Save to `notes/session-log-YYYY-MM-DD.md`.
